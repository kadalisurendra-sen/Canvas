# US-017: Master Data Categories Listing

**As an** admin
**I want to** see all master data categories in a sidebar with item counts
**So that** I can navigate between different data categories to manage their values

## Acceptance Criteria

- [ ] AC1: Master Data Management page at `/master-data` displays a left sidebar listing all categories
- [ ] AC2: `GET /api/v1/master-data/categories` returns all categories with fields: id, name, display_name, icon, sort_order, item_count (count of active values)
- [ ] AC3: Sidebar shows each category with its Material icon, display name, and item count badge (e.g., "Organizational Taxonomy (128)")
- [ ] AC4: Categories listed include: Organizational Taxonomy, KPIs, Solution Types, Digital Platforms, ML Models, Risk Categories
- [ ] AC5: Clicking a category loads its values in the main content area (right panel) — connects to US-018
- [ ] AC6: The first category is selected by default on page load
- [ ] AC7: Active category is highlighted with a left border accent and background tint

## Technical Notes
- Categories are seeded into the `master_data_categories` table during tenant DB initialization
- Sidebar uses the same dark navy style as the main navigation sidebar for consistency
- Item count is computed via a COUNT subquery on `master_data_values` where `is_active = true`

## Dependencies
- US-001 (Tenant Database schema)
- US-025 (Shared Layout)
