# US-022: Analytics Dashboard — Charts

**As an** admin
**I want to** see visual charts showing evaluation trends, template usage, and stage distribution
**So that** I can analyze patterns and make data-driven decisions

## Acceptance Criteria

- [ ] AC1: Bar chart "Use Cases by Stage" shows counts for each stage: Desirable, Feasible, Viable, Prioritized, Not Started; bars are colored with the primary purple palette
- [ ] AC2: Pie/donut chart "Template Usage Distribution" shows percentage breakdown by template category: AI/ML, RPA, Agentic AI, Data Science, Custom
- [ ] AC3: Line chart "Evaluations Over Time" shows monthly evaluation count trend over the selected date range
- [ ] AC4: A "Top Users Activity" table below the charts shows columns: User, Evaluations Count, Last Active; limited to top 10 users
- [ ] AC5: `GET /api/v1/analytics/dashboard` response includes chart data: `stage_distribution`, `template_usage`, `evaluations_timeline`, and `GET /api/v1/analytics/top-users` returns the top users list
- [ ] AC6: Charts are responsive and resize correctly within the content area
- [ ] AC7: Charts update when the date range filter changes (from US-021)
- [ ] AC8: Empty states: charts show "No data available" message when there's no data for the selected period

## Technical Notes
- Use Recharts library for all charts (consistent with scoring chart in US-011)
- Chart data is pre-aggregated by the backend to minimize client-side computation
- Top Users table reuses the same table component styles from the design system

## Dependencies
- US-021 (Dashboard Metrics — provides the page structure and date filter)
