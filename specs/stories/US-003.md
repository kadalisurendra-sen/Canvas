# US-003: Login UI Page

**As a** user
**I want to** see a login page with username/password form, remember me, forgot password, and register links
**So that** I can authenticate and access the admin panel

## Acceptance Criteria

- [ ] AC1: Login page renders at route `/login` with a centered card layout matching the design system (white card, light gray background #F7F8FC)
- [ ] AC2: Form contains username input, password input, "Remember me" checkbox, and a "Sign In" submit button (primary purple #5F2CFF)
- [ ] AC3: "Forgot Password?" link redirects to the Keycloak password reset flow URL
- [ ] AC4: "Register" link redirects to the Keycloak self-registration URL
- [ ] AC5: On successful sign-in, the user is redirected to the main dashboard (Template Management page)
- [ ] AC6: On authentication failure, an error message is displayed below the form ("Invalid username or password")
- [ ] AC7: Form validates that username and password are non-empty before submission; shows inline validation errors
- [ ] AC8: "Helio Canvas 2.0" branding with logo is displayed above the login form
- [ ] AC9: Font is Montserrat throughout the page

## Technical Notes
- React Router guards redirect authenticated users away from `/login` to the dashboard
- Use React Query mutation for the login API call
- Store JWT in httpOnly cookie (handled by backend callback)
- The "Remember me" checkbox sets a longer session TTL via a query parameter to the auth endpoint
