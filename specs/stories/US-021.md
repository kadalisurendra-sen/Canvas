# US-021: Analytics Dashboard — Metric Cards

**As an** admin
**I want to** see key performance indicators on the analytics dashboard
**So that** I can quickly understand the current state of evaluations in my tenant

## Acceptance Criteria

- [ ] AC1: Analytics page at `/analytics` displays two tabs: Dashboard, Audit Log; Dashboard is active by default
- [ ] AC2: Dashboard tab shows a date range filter (dropdown: Last 7 Days, Last 30 Days, Last 90 Days, Last Year, Custom); defaults to "Last 30 Days"
- [ ] AC3: "Export Report" button is displayed at the top right (functionality in US-024)
- [ ] AC4: `GET /api/v1/analytics/dashboard?from={date}&to={date}` returns aggregated metrics
- [ ] AC5: KPI metric cards displayed in a row: Total Projects (count), Total Use Cases (count), Active Evaluations (count), Completed Evaluations (count), Average ROI (percentage)
- [ ] AC6: Each metric card shows: icon, metric label, value (large text), and percentage change from previous period (green up arrow or red down arrow)
- [ ] AC7: Cards use the design system: white bg, rounded-xl, shadow-sm, primary text for value
- [ ] AC8: Metrics update when the date range filter changes

## Technical Notes
- Backend aggregates data from tenant evaluation tables (may need mock/seed data initially if evaluation tables don't exist yet)
- Percentage change calculated by comparing current period to the equivalent previous period
- React Query with the date range as a query key for automatic refetching

## Dependencies
- US-001 (Tenant Database)
- US-025 (Shared Layout)
